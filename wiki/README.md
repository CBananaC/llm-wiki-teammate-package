# LLM Wiki for the 林爽文事件 Project

This wiki is the project’s compact research-memory layer. It keeps current research context, source rules, extraction rules, workflows, and links to canonical skills.

Start with [[index]]. Operational state belongs in `PROJECT_LOG.md`; source data and review state belong in `review-tools/`; reusable prompts belong in `tool/skills md/`.

## Core rules

- Preserve source wording and provenance.
- Separate source evidence, external enrichment, and interpretation.
- Mark uncertainty instead of guessing.
- Use English by default and Traditional Chinese when Chinese is needed.
- Do not store credentials in the wiki.
